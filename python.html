Python 뿌시기

The Python

파이썬은 프로그래밍 언어 중 하나일 뿐이지만 들여다보면 끝없이 넓은 바다와 같은 존재이다. 놀랍도록 풍부한 라이브러리와 다양한 용도로 인해 그 어떤 프로그래밍 언어들보다 명확한 가치를 인정받았다. 간결하고 명료한 그 문법은 많은 개발자에게 사랑받는 이유 중 하나이다. 또한, 파이썬의 시작은 프로그래밍 입문자들에게 진입장벽이 더 낮은 언어로 인식되어져 있다. 이로 인해 다양한 산업 분야에서 활용되며 이를 통해 데이터 분석, 웹 개발, 머신러닝 및 인공지능 개발 등 많은 코딩 활동에 사용되게 되었다. 워낙 다목적으로 쓰이기 때문에 정말 다양한 분야에 파이썬의 발자취를 찾아볼 수 있다 물론, 파이썬만의 강점 뿐 아니라 같은 프로그래밍 원칙에 맞춰 작동하는 다른 여러 프로그래밍 언어들도 경쟁력을 갖추고 있지만, 파이썬은 그 중에서도 독보적인 강점으로 자리잡았다. 세계적으로도 인기 있는 파이썬은 프로그래머들에게 알맞은 언어를 찾기 위한 중요한 바로미터로 쓰인다. 결론적으로, 파이썬은 무한한 가능성을 가진 실용적이고 다재다능한 프로그래밍 언어이다. 그것은 디지털 세계의 파도가 되어 바뀌기를 거부하는 것처럼, 프로그래밍에 변화를 가져 왔다. 비록 나도 코딩에 있어 초보자이지만 여기에 다시겸 노트 정리와 같이 함으로써 파이썬에 대한 지식을 키워보려고 한다.

Beginning Setup

파이썬 내용에 관한 페이지이기에 초기 설정 및 베이스 Development에 대해서 자세히 다루지는 않겠지만, python 코딩을 할 때 PyCharm을 통해 코딩하는 것이 가장 편하다고 꼽을 수 있다. PyCharm은 JetBrains에서 만든 인기있는 Python 통합 개발 환경(IDE)이다. PyCharm을 구글에 검색해 들어가면 다운로드할 수 있는 링크가 있다. Community와 Professional 두 가지 버전 중 고를 수 있는데, Community 버전은 오픈 소스 프로젝트에 사용 가능하며 무료로 다운로드 받을 수 있다. 반면 Professional 버전에는 추가적인 고급 기능들이 있지만, 유료로 사용해야 한다. 입문자 및 학생에게 있어 Community 버전만 해도 충분한 기을 제공하므로, 무료로 사용 가능한 이 버전으로 시작하는 것이 좋다. PyCharm 버전을 설치하고 나면, 프로젝트 생성 및 관리, 코드 편집 및 실행, 디버깅 등 기본적인 Python 개발 작업을 수행할 수 있다. 또한, 이 IDE는 다양한 플러그인과 확장 기능을 지원하여 개발 경험을 향상시키기에 좋다. 기본 설정 외에도, 사용자별 선호에 따라 다양한 추가 기능을 사용할 수 있다. 결론적으로, PyCharm Community 버전은 파이썬을 배우고 사용하기 시작하는 사람들에게 이상적인 개발 도구이며, 이를 통해 Python 기초 및 다양한 프로젝트를 수행하기에 좋은 출발점이 될 것이다.

Chapter 1: Data Types and Variables

  1. 숫자형(Numeric types): 중요특징: Not Iterable = 반복이 불가능하다(loop에 못돌린다는 의미)
    • Integer(정수형): 1, 30, 101와 같이 소수점 없는 숫자 ex) a = 1, b = 77
    • Float(실수형): 소수점 있는 숫자들 1.43, 3.141592, 0.111111. They can represent decimals and are useful for various mathematical operations. ex) PI = 3.14, RTWO = 1.414
    • Complex(복소수형): 수학시간에 배우는 실수와 허수부분을 가지는 복소수 표현. 일반적으로 a + b j 형태로 나타낼수있다. 사실 나도 아직까지 코딩에서 써본적이 없다.
  2. Bool: True or False를 값으로 가지는 데이터타입(0은 보통 Fasle로 간주, 나머지 0이 아닌 수나 문자 모두 True) ex) Name = True
  3. Sequence types: 중요특징: Iterable = 반복 가능하다 (loop에 돌릴 수 있다는 의미)
    • String " " : 문자열(String)은 일련의 문자들로 구성된 불변하는 데이터 타입으로, 파이썬에서 작은따옴표(') 또는 큰따옴표(")로 둘러싸인 텍스트. 아주아주 많이 쓰이며 이 자료형은 Imutable이다. ex) my_str = "Hello World"
    • List [ ] :리스트는 순서가 있는 변경 가능한 요소들의 모음으로, 여러 데이터 타입을 포함할 수 있는 파이썬의 자료형. 엄청 자주 쓰이며 Mutable의 대표주자. append, insert, remove 등등 함수로 변형가능. ex) my_list = [1, 2.22, 3, "a", 'b', False]
    • Tuple ( ) : 튜플은 순서가 있으며 변경 불가능한 요소들의 모음으로, 여러 데이터 타입을 포함할 수 있는 파이썬의 자료형. 리스트와 비슷해보이지만 튜플은 Immutable의 대표주자. 한번 생성 후 변경이 절대로 NO. ex) my_tuple = (1,2)
  4. Set { }: 세트(Set)는 순서 없고 중복된 요소를 허용하지 않는, 변경 가능한 파이썬 자료형. 수학의 집합을 생각하면 된다. {1, 1, 1, 2, 3} 이나 {1, 2, 2, 2, 3}이나 중복된 값를 지우고 똑같이 {1, 2, 3}이 된다.
  5. Dictionaries {Key : Value} : 딕셔너리(Dictionary)는 키(Key)와 값(Value) 짝으로 이루어진, 순서가 없는 변경 가능한 파이썬 자료형. 키를 통해 쉽게 값에 접근할 수 있어 데이터 관리에 유용한 구조. 키와 값이 짝으로 되어있는 것을 우리는 매핑(mapping)이라고 한다. 공부하면서 가장 어려웠던 데이터형. 자주 같이 쓰이는 함수로 values(), keys(), enumarate() 등등 있다. ex) My_dict = {"Name": "Jun", "Age" : 25, "Married" :False} => 딕셔너리이름[키]로 값을 찾을 수 있다.
  • Assigning values to variables and basic arithmetic operations - 파이썬 속 사칙연산
    • Addtion(+): a = 3, b = 5, c = a + b => c? = 8
    • Subtraction(-): x = 10, y = 3.4, z = x - y => z? = 6.6
    • Multiplication(*): x = "HI", y = 3, z = x * y => z? = "HIHIHI"
    • Division(/): a = 10, b = 5.0, c = a / b => c? = 2.0
    • Floor division(//): a = 10, b = 3, c = a // b => c? = 3
    • Modulus(%): a = 10, b = 3, c = a % b => c? = 1
    • Exponentiation(**): a = 2, b = 4, c = a ** b => c? = 16
    • (추가사항) Multiplication에서는 숫자끼리 곱셈도 가능하지만 예시와 같이 string형과 숫자형 사이의 곱셈도 가능하다
    • (추가사항) Division에서 0으로 나누었을 때 파이썬은 ZeroDivisionError라는 오류메세지를 띄운다
  • Frequently Used Operators - 코딩할때 많이 쓰이는 기호들
    • x += 1 => x = x + 1 (use in for loop)
    • x -= 3 => x = x - 3
    • == : x == y Equal =기호 하나만 사용하면 값을 지정 두개 사용하면 양변이 같은지 boolean으로 대답
    • != : 코딩에서 !는 아니다를 의미한다 ==의 반대
    • >, <, >=, <= : 양변 비교하는 기호들. 모두 True or False로 return함
    • and : Returns True if both statements are true => x < 5 and x < 10
    • or : Returns True if one of the statements is true => x < 5 or x < 4
    • not : Reverse the result, returns False if the result is true => not(x < 5 and x < 10)
    • is : Returns true if both variables are the same object => x is y
    • in : Returns True if a sequence with the specified value is present in the object. 리스트나 딕셔너리 안에 원하는 값이 있는지 없는지 확인할때 자주 사용됨 => x in y

Chapter 2: Output/Input/Comments

Input Example:

<1. Get user input >
user_name = input("Enter your name: ")
<2. Display the entered name >
print("Hello, " + user_name + "!")

Output Example:

From the upper input example code,
If I typed "Jun" as a user name
Output that will be shown from the print code is
Hello, Jun!

"Input": 컴퓨터가 사용자로부터 받는 값
"Output": 컴퓨터가 사용자로부터 받은 데이터를 프로그램 속 작업에서 실행 후 다시금 사용자가 읽을 수 있도록 반환하는 값
"Comments"(주석): 실질적으로 프로그래밍에서 효과는 없지만 코딩을 하는데 있어 노트같은 역활. Using comments to explain and document code Overview of the print() function for displaying output Formatting output with string interpolation (f-strings, format() method, etc.) Introduction to the input() function for gathering user input Example program that demonstrates using input to interact with users
파이썬에서 주석 처리는 단일 문장과 문단을 사용할때 다른 코드가 필요하다.
i) 한줄 주석은 #을 사용. ex) #This is a comment
ii) 문단 주석은 """ """을 사용 double quotation mark를 양쪽에 3개 사용함으로 저 사이에 있는 내용은 모두 주석 처리된다. ex) """Here goes the paragraph"""
